
# 🎤 Shout-Out Voice Assistant

## 🚀 Overview

- An interactive Python project that uses Windows Speech API (SAPI) to deliver fun, personalized shout-outs. Input names, choose a voice, and let the assistant announce them aloud with enthusiasm.

## ✨ Features

- 🎙️ Text-to-Speech powered by Windows SAPI

- 🔄 Randomized prompts for engaging interaction

- 🗣️ Multiple voice options to choose from

- 📢 Energetic shout-outs for any list of names

- 🛑 Exit option for smooth program termination

## 🛠️ Tech Stack

- Python

- pywin32

### ▶️ How to Run

```
pip install pywin32
python shoutout.py
```

## 🧠 Example

- Select voice index: 1
- 👉 Who deserves a shout-out today? (type 'exit' to quit): John, Alice
- Shout-out to John
- Shout-out to Alice
